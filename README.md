# Escaper
Escaper [Netlify 주소](https://escaper.netlify.com/)
원래는 EPub Shortcut Creator인데요 줄여서 ESC로 하려다가 이렇게 됐네요 하하
사용 기술: Ant design + Vue(cli) + Typescript
사용 라이브러리:
  - JSZip(Epub 생성)
  - file-saver(파일 저장)

npm install
